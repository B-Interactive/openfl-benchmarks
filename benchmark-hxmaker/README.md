# benchmark-hxmaker

Hxmaker is still in the development phase, so it is necessary to install the required dependencies through the following methods:

```shell
# hxmaker
haxelib git hxmaker https://github.com/rainyt/hxmaker
haxelib git hxmaker-openfl https://github.com/rainyt/hxmaker-openfl
# other
haxelib install openfl
haxelib openfl setup
haxelib install openfl-glsl
haxelib install actuate
haxelib install vector-math
```
